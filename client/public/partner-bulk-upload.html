JSON Format (minimum required fields — up to 50 courses per upload):

[
  {
    "title": "Introduction to CBT",
    "description": "A foundational overview of Cognitive Behavioral Therapy techniques.",
    "ceHours": 1.5
  },
  {
    "title": "Trauma-Informed Care",
    "description": "Evidence-based approaches to working with trauma survivors.",
    "ceHours": 3,
    "objectives": ["Identify trauma responses", "Apply grounding techniques"],
    "categories": ["Trauma"],
    "tags": ["TIC", "PTSD"],
    "status": "draft"
  }
]

Only title, description, and ceHours are required. All other fields are optional.
